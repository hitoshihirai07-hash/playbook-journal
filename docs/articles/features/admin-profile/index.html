自己紹介管理者

好きなポケモンはピカチュウ

好きなポケモンはピカチュウです。昔から分かりやすく親しみやすい存在で、ポケモンをあまり知らない人にも伝わる強さがあります。

このサイトでも、最初から難しい話を詰め込みすぎるのではなく、ピカチュウのように入口として分かりやすい記事を増やしていきたいと考えています。

好きな世代はハートゴールド・ソウルシルバー

好きな世代のゲームは、ハートゴールド・ソウルシルバーです。ジョウト地方だけでなくカントー地方も冒険でき、16個のバッジを集められるところが特に好きです。

さらに、バトルフロンティア系のやり込み要素があるので、クリア後も長く遊べるのが魅力です。旅を進める楽しさと、あとからじっくり挑戦する楽しさの両方がある作品だと思っています。

好きな球団は巨人

好きな球団は巨人です。選手ごとの役割や、シーズンの中で調子や起用法が変わっていくところを見るのが好きです。

OBで好きな選手は高橋由伸です。現役選手では、リチャード、小林誠司、坂本勇人が好きです。数字だけではなく、守備、役割、勝負どころでの印象も含めて見ていきたいタイプです。
