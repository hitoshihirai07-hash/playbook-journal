プロ野球 投手成績入門まとめ：防御率から救援指標まで

2026年6月4日 公開 ・ 2026年6月6日 更新 ・ 5分

投手成績は数字が多く見えますが、最初から全部を覚える必要はありません。防御率、WHIP、自責点、奪三振率を順番に見るだけでも、投手の特徴はかなり読みやすくなります。

まとめプロ野球

まずは失点を防ぐ力を見る

最初の入口は防御率です。投手がどれくらい失点につながる場面を抑えているかを大まかに確認できます。

  • 防御率とは？ 投手の成績を見るときに最初に覚えたい数字 →
  • 自責点と失点の違い →

次に走者を出しにくいかを見る

防御率だけでは、毎回走者を出しているのか、テンポよく抑えているのかまでは分かりません。そこでWHIPを見ると、安定感を補って確認できます。

  • WHIPとは？ 投手の安定感をシンプルに確認する →

三振を取れる投手かを見る

奪三振率を見ると、投手がどれくらい自分でアウトを取れるタイプなのかを考えやすくなります。ピンチで三振を取れるか、打たせて取るタイプかを見る入口になります。

  • 奪三振率とは？ 投手の特徴を見るときに便利な数字 →

先発投手が試合を作れたかを見る

QSを見ると、先発投手が勝ち負けとは別に、試合の土台を作れたかを確認しやすくなります。防御率や投球回と合わせて見ると、先発投手の安定感を考えやすくなります。

  • QSとは？ 先発投手が試合を作る意味を知る →
  • 先発ローテーションの見方も確認する →

救援投手の役割を見る

救援投手を見るときは、セーブとホールドの違いを知ると役割を整理しやすくなります。試合を締める投手と、終盤へつなぐ投手では見たい数字が少し変わります。

  • セーブとホールドの違い：救援投手の役割を整理 →

今後増やすとよい投手成績

  • 投球回：防御率やWHIPの安定度を見るときに必要です。
