4倍弱点から覚える複合タイプ入門：ポケモンごとに整理する

2026年6月2日 ・ 5分

二つのタイプを持つポケモンは、弱点が重なって4倍になることがあります。最初は、よく見るポケモンの4倍弱点から覚えると整理しやすくなります。

ゲーム入門タイプ相性

二つの弱点が重なると4倍になる

例えば、ある技が一つ目のタイプにも二つ目のタイプにも効果抜群になる場合、タイプ相性上の効果は4倍になります。

相性表を丸ごと覚える前に、ポケモン名、通常のタイプ、4倍弱点をセットで確認しましょう。

最初に覚えやすい4匹

  • ガブリアス：ドラゴン・じめんタイプ。こおりタイプの技が4倍。
  • ギャラドス：みず・ひこうタイプ。でんきタイプの技が4倍。
  • リザードン：ほのお・ひこうタイプ。いわタイプの技が4倍。
  • ハッサム：むし・はがねタイプ。ほのおタイプの技が4倍。
ここでは通常のタイプを前提に整理しています。実際の対戦では、タイプを変える仕組み、特性、持ち物、技なども確認しましょう。

自分のチームから覚える

  • 自分がよく使うポケモンに4倍弱点があるか確認する。
  • 対戦でよく見かけるポケモンを一匹だけ追加する。
  • 4倍弱点がないポケモンも、苦手なタイプを一つずつ確認する。
  • 慣れてきたら相性表を使い、覚えていない組み合わせを補う。

4倍弱点だけで勝敗は決まらない

4倍弱点を覚えると行動を考えやすくなりますが、それだけで勝敗が決まるわけではありません。素早さ、技を使えるかどうか、交代、特性なども関係します。

まずは相手の大きな弱点に気づけるようになり、慣れてから他の判断材料を増やしましょう。
