プロ野球の順位表の見方：勝率・ゲーム差・貯金をやさしく読む

2026年6月7日 ・ 5分

プロ野球の順位表には、勝ち数や負け数だけでなく、勝率、ゲーム差、貯金などの言葉が並びます。意味が分かると、チームの状態を追うのがぐっと楽しくなります。

観戦入門順位表プロ野球

順位は勝率で並ぶ

プロ野球の順位表では、基本的に勝率が高いチームほど上に並びます。勝率は、勝った試合の割合を見る数字です。

勝ち数だけを見ると、試合数が違うチーム同士を比べにくい場合があります。そのため、順位表では勝率が大切になります。

ゲーム差は首位との距離

ゲーム差は、上の順位のチームとどのくらい離れているかを見る目安です。数字が小さいほど追いつきやすく、大きいほど差があると考えます。

ただし、ゲーム差だけで今後を決めつける必要はありません。直接対決の残り試合、連勝や連敗、投手陣の状態などでも印象は変わります。

貯金と借金は勝ち越しの数

貯金は勝ち数が負け数より多い状態、借金は負け数が勝ち数より多い状態を表す言葉です。たとえば20勝15敗なら、貯金5です。

貯金があるチームは安定して勝ち越していると見やすく、借金があるチームはまず勝率5割を目指す見方になります。

順位表は流れを見る入口

  • 最近の勝敗を見る。順位が同じでも、上向きか下向きかが分かります。
  • 直接対決を見る。近い順位のチーム同士の試合は差が動きやすくなります。
  • 投手の登板予定を見る。先発投手の状態で試合の見え方が変わります。
